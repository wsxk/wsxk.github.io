## 2. dir/help<br>
**dir用于查看一个库中有哪些组件，help用于查看组件的具体用法**<br>
```python
import torch
if __name__ == '__main__':
    print(dir(torch))
    print((dir(torch.cuda)))
    print(help((torch.cuda.is_available)))
```
