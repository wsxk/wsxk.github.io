# 5. 为docker镜像安装pwndbg调试环境<br>
![](https://raw.githubusercontent.com/wsxk/wsxk_pictures/main/2025-9-25/20260413001645.png)
[https://pwndbg.re/stable/setup/#portable-release](https://pwndbg.re/stable/setup/#portable-release)<br>
下给出了一键式安装的步骤,仅需要在dockerfile中添加如下内容:<br>
```
RUN apt-get install -y lib32z1 libevent-2.1-7 libseccomp2 git gdb curl xz-utils
RUN curl --proto '=https' --tlsv1.2 -LsSf 'https://install.pwndbg.re' | sh -s -- -t pwndbg-gdb
```
启动容器时，记得添加`ptrace`权限:<br>
```
docker run --cap-add=SYS_PTRACE -d -p "0.0.0.0:8888:8888" -p "0.0.0.0:8889:8889/udp"  --restart always suctf_evbuffer:pwn1

docker exec -u 0 -it IMAGE_ID bash
```
