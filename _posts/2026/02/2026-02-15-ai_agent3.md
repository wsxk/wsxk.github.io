# 1. openclaw<br>
当红炸子鸡，据说非常强力，官网链接:[https://openclaw.ai/](https://openclaw.ai/)<br>
在linux里安装比较好（ubuntu 24.04）最佳，因为官方也这么干~<br>
## 1.1 openclaw 安装过程<br>
安装过程:<br>
```
1. curl -fsSL https://openclaw.ai/install.sh | bash  # 执行安装脚本，不出意外，安装成功后会自动执行第2步
2. openclaw onboard  # 设置一些基本配置
```
## 1.2 配置模型<br>
配置模型时，我选择的是openai的认证，选择后，openclaw会给你一个认证链接，完成认证后将认证成功的url返回即可完成认证。<br>

## 1.3 openclaw 配置chrome浏览器<br>
```
openclaw browser --browser-profile openclaw status
openclaw browser --browser-profile openclaw start
openclaw browser --browser-profile openclaw open https://example.com
openclaw browser --browser-profile openclaw snapshot

openclaw browser extension install # 生成chrome插件
```
按照openclaw给出的提示执行:<br>
![](https://raw.githubusercontent.com/wsxk/wsxk_pictures/main/2025-9-25/20260208235330.png)
