# 1. M0、M1、M2<br>
**M0、M1、M2都是衡量社会中“钱有多少”的指标，区别主要在于这些钱的流动性：越容易马上拿来支付，层级越靠前。**<br>
```
M0: 手里的现金，通常指社会公众持有的纸币和硬币
M1（狭义货币）：随时能花的钱， M0+ 活期存款
M2（广义货币）：社会上大部分钱，M1 + 定期存款、储蓄存款等
```
`M2-M1` 通常被称为准货币，准货币增长过快，说明企业和居民都更倾向于储蓄。<br>


# 2. 国债<br>
国债有国债利率、国债价格、国债收益率3个指标。为了解释他们之间的区别，需要解释一级市场和二级市场的概念。<br>
美国国债市场分为一级市场和二级市场。**美国国债一级市场为发行市场，主要针对的是机构和零售投资者（竞争性参与者和申购额度被限制的非竞争性参与者），像各国央行购买美国国债，基本上都是在一级市场参与美国国债的拍卖。**<br>
央行在一级市场竞拍到美国国债以后，将来要在二级市场抛售。所以，美国国债二级市场是极其活跃的，流通性非常好。对普通个人而言，都是在二级市场购买国债<br>
```
国债利率：国债在政府发行时确定的利率，比如政府发现了一张1000美元的国债，为期三年，国债利率5%，即每年固定支出1000* 5% = 50美元。
国债价格：政府发行的国债通常是一级市场，其他央行或者美联储买了国债后，可以选择放到二级市场销售，二级市场销售的价格就被称为 国债价格，国债价格可以是995美元，1000美元、1005美元等。
国债收益率： 对个人而言，你如果以 800美元买入一张国债，持有一年后，以900美元的价格卖出，对你而言，净收益（-800+50+900=150）美元，实际国债收益率为 150/800 =  18.75%
```
