# 1. M0、M1、M2<br>
**M0、M1、M2都是衡量社会中“钱有多少”的指标，区别主要在于这些钱的流动性：越容易马上拿来支付，层级越靠前。**<br>
```
M0: 手里的现金，通常指社会公众持有的纸币和硬币
M1（狭义货币）：随时能花的钱， M0+ 活期存款
M2（广义货币）：社会上大部分钱，M1 + 定期存款、储蓄存款等
```
`M2-M1` 通常被称为准货币，准货币增长过快，说明企业和居民都更倾向于储蓄。<br>
