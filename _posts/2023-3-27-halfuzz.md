## install<br>
**环境 ubuntu18.04**<br>
### 安装python2<br>

    apt install python

### 安装python3虚拟环境<br>

    apt install python3-venv

### 使用python3虚拟环境进行安装<br>

首先从github上clone hal-fuzz下来，然后

    python3 -m venv hal-fuzz-python
    source hal-fuzz-python/bin/activate
    sudo ./setup.sh
